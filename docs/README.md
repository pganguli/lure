# Docs

## Building documentation
Lure uses Sphinx for autogenerated documentation. To build the docs, first install Sphinx:

    pip install -U sphinx sphinx_mdinclude

Then in the `docs/` directory, run:

    sphinx-apidoc -o source ../src/
    make html

The docs will be in `docs/build/html`.

## Writing documentation

### Tips
For referencing python objects within a docstring ensure you use back quotes (\`) not regular quotes (').
For example, to reference the `sensor_nody.py` module, write:
    
    :py:mod:`lure.node.sensor_node`
